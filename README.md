<!-- Prerequisites -->

### :bangbang: Prerequisites

<!-- Installation -->

### :gear: Installation

Install my-project with npm

```bash
  npm install
  cd task-tracking-app
```

<!-- Running Tests -->


### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/leaser019/Task-Tracking-Application
```


Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
